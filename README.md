## Summary ##
Snips saves your code while offering fast copy-paste, live searching, and simple organization.

## Features ##
* Syntax highlighting
* One-click copy and pasting
* Live-search

## Getting Started ##
From the root directory, run:

```

npm install
cd public
bower install
cd ..
node server.js

```

Then visit localhost:8000 in any browser.
